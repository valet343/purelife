# Швидкий старт - Git Workflow

## Щоденна робота

### 1. Робіть зміни локально
Редагуйте файли як завжди.

### 2. Використовуйте скрипт для деплою

**PowerShell:**
```powershell
.\deploy.ps1 "Опис ваших змін"
```

**Git Bash / Linux:**
```bash
chmod +x deploy.sh
./deploy.sh "Опис ваших змін"
```

### 3. Підтвердіть відправку
Скрипт запитає підтвердження перед відправкою на сервер.

## Або вручну:

```powershell
# 1. Перевірити зміни
git status

# 2. Додати зміни
git add .

# 3. Створити коміт
git commit -m "Опис змін"

# 4. Відправити на сервер
git push origin main
```

## Налаштування

- ✅ Remote налаштовано: `origin` → GitHub
- ✅ Pre-push hook активний (перевіряє конфіденційні файли)
- ✅ .gitignore налаштовано (config.php виключено)

## Важливо

⚠️ **НЕ** комітьте файли:
- `config.php`
- `admin/config.php`
- Логи та кеш

Вони автоматично виключені через `.gitignore`.
